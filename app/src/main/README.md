BarcodeScanner
==============

Android手机客户端关于二维码扫描的源码，使用了zxing 3.1.1代码并对其进行了精简，支持低版本的sdk，实现了二维码和一维码的扫描、从图片解析一维码和二维码，闪光灯、调焦。。。
